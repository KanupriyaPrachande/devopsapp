# ✨ Overview

NebulaOps is a modern DevOps dashboard designed to simulate real-world infrastructure monitoring systems used by cloud and platform engineering teams.

The project combines:

- Real-time monitoring UI
- Terminal-inspired interface (TUI aesthetics)
- JWT Authentication
- Backend API integration
- Container monitoring simulation
- Cloud deployment practices
- Full-stack architecture

This project was built to showcase practical DevOps + Full Stack + Cloud Engineering skills in a visually unique way.

# 🧠 Why This Project Is Different

Most monitoring dashboards focus only on functionality.

NebulaOps focuses on:

- Developer experience
- Terminal-inspired UI
- Real-world DevOps concepts
- Cloud deployment workflow
- Full-stack integration
- Modern infrastructure visualization

The goal was to build something that feels like an actual operations console rather than a standard admin dashboard.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/KanupriyaPrachande/NebulaOps.git
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
